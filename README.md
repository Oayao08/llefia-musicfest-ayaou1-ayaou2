# Llefià Music Fest
La pàgina web conté els navegadors a cada arxiu, i té diferents etiquetes. Cada pàgina segueix un ordre de:
  - Header
  - Main
  - Footer

Estructura de fitxers:
llefia-musicfest-cognom1-cognom2/
├── index.html
├── style.css
├── README.md
├── escenari.jpg
└── paginas/
    ├── cartel.html
    ├── horario.html
    └── contacto.html

Instruccions d'ús:
  - Localitzar carpeta del projecte
  - Buscar fitxer principal
  - Obrir arxiu index.html
  - Visualització del lloc web
  - Veure si la web té més pàgines a <nav>
  - Clicar els diferents menús del navegador segons desitg de l'usuari
